# BMI Calculator

A simple Android Studio BMI Calculator project built with Java.

## Features

- Enter weight in kilograms.
- Enter height in centimeters.
- Calculate BMI.
- Show BMI category.
- Developed by Jakariya Talukdar

## Open in Android Studio

1. Open Android Studio.
2. Choose **Open**.
3. Select this project folder.
4. Let Gradle sync.
5. Run the app on an emulator or Android device.
